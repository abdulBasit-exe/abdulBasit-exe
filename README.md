## Featured Projects

### 🔷 [NYC Taxi Data Engineering Pipeline](https://github.com/abdulBasit-exe/NYC24-AzureDE-Project)
End-to-end pipeline on Azure processing real-world NYC Taxi trip data. Implements Medallion Architecture (Bronze → Silver → Gold) using ADF for dynamic ingestion via HTTP API, Databricks with PySpark for transformation, Delta Lake for versioned storage, and aggregated analytical datasets queryable with time travel.  
`Azure Data Factory` `Databricks` `Delta Lake` `PySpark` `ADLS Gen2`

### 🔷 [YouTube Trending Data ETL — AWS](https://github.com/abdulBasit-exe/youtube-etl-project)
Serverless ETL pipeline on AWS that ingests YouTube trending video data from Kaggle, processes and converts it from CSV to Parquet using AWS Glue, triggers transformations via Lambda, and surfaces insights through Athena queries and QuickSight dashboards.  
`AWS Glue` `Lambda` `S3` `Athena` `QuickSight` `PySpark`

### 🔷 [Adventure Works Azure Data Engineering](https://github.com/abdulBasit-exe/AW-Azure-Data-Engineering-Project)
Full Azure stack pipeline ingesting AdventureWorks data from GitHub via dynamic ADF pipelines with parameterized Lookup and ForEach activities. Transforms data in Databricks using PySpark, writes to ADLS Gen2, and exposes Gold-layer views through Synapse Serverless SQL for downstream reporting.  
`Azure Data Factory` `Databricks` `Synapse Analytics` `ADLS Gen2` `PySpark`

### 🔷 [Tokyo Olympic Data Pipeline](https://github.com/abdulBasit-exe/Tokyo-Olympic-Data-Pipeline)
End-to-end data engineering pipeline on Azure using ADF for ingestion, Databricks for transformation, and structured storage for analysis of Tokyo 2020 Olympics data.  
`Azure Data Factory` `Databricks` `ADLS Gen2` `PySpark`
